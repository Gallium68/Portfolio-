## 📌 About
This is my personal portfolio website showcasing my skills, projects, and professional background as a Full Stack Developer.

## ✅ Features
- Responsive design (works on mobile & desktop)
- Animated hero section with professional typography
- Skills & tech stack section with progress bars
- Projects section with live links
- Timeline-based experience & education section
- SEO-friendly meta tags (Open Graph, keywords, description)
- Smooth scroll navigation

## 🛠️ Tech Stack
- HTML5
- CSS3 (custom properties, animations, grid, flexbox)
- Vanilla JavaScript (Intersection Observer, smooth scroll)
- Google Fonts (Syne, DM Sans, Josefin Sans)

## 🚀 Setup & Deployment

### Run Locally
```bash
# Just open in browser — no build step needed
open index.html
```

### Deploy on GitHub Pages
1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to **main branch**
4. Your site will be live at `https://Gallium68.github.io/portfolio`
